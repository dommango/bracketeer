Pitch-green pill button — the workhorse CTA. Use `primary` for advancement actions (save, submit), `gold` for celebratory winning actions, `secondary` for cancel/secondary paths.

```jsx
<Button variant="primary" size="lg" block>Save my bracket</Button>
<Button variant="gold">Lock it in</Button>
<Button variant="secondary">Cancel</Button>
<Button variant="ghost" size="sm">Skip</Button>
```

Variants: `primary` · `secondary` · `gold` · `ghost` · `danger`. Sizes: `sm` (32px), `md` (40px), `lg` (52px — kickoff submit). Always pill-shaped. Active state shrinks 3% for tactile feedback.
