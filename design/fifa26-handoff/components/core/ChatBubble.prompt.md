A single chat message bubble. `mine` flips it to the right and uses pitch-green; otherwise it's a paper-sunk bubble with the author name colored by `authorColor`.

```jsx
<ChatBubble body="kane goal — that's +3" authorName="Sam" authorColor="city-philadelphia" timestamp="2026-06-27T19:14:00Z" />
<ChatBubble body="cope" mine timestamp="2026-06-27T19:14:30Z" />
```
